# [DEPRECATED] AgentOps Toolkit Skills

> ⚠️ **This extension has been renamed.** It is now published as **"AgentOps Accelerator — Skills for GitHub Copilot"** under the `AgentOpsAccelerator` publisher. This `AgentOpsToolkit` build no longer ships features and exists only to prompt migration.

## Migrate in two steps

1. **Install the new extension** from the Marketplace: [AgentOpsAccelerator.agentops-accelerator](https://marketplace.visualstudio.com/items?itemName=AgentOpsAccelerator.agentops-accelerator).
2. **Uninstall this extension** (`AgentOpsToolkit.agentops-toolkit`) from your Extensions view.

## Why the rename?

The project rebranded from "AgentOps Toolkit" to "AgentOps Accelerator" to better reflect its scope. The Marketplace requires a new publisher identifier for the rename, so the extension was republished under `AgentOpsAccelerator`. See the tracking issue for full context: <https://github.com/Azure/agentops/issues/181>.
